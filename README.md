# 🌐 Portfolio Website Auditor

A modern portfolio website auditing tool built with HTML, CSS, and JavaScript. Analyze portfolio websites for SEO, performance, mobile responsiveness, and UI/UX through an interactive dashboard.

---

## 🚀 Features

### 🔍 Website Audit

- Portfolio URL Input
- Website Analysis Dashboard
- Instant Audit Results

### 📈 SEO Score

- SEO Rating Display
- Optimization Suggestions
- Metadata Recommendations

### ⚡ Performance Score

- Website Performance Rating
- Speed Improvement Suggestions
- Optimization Checklist

### 📱 Mobile Friendly Check

- Mobile Responsiveness Score
- Mobile Optimization Tips

### 🎨 UI / UX Analysis

- Design Quality Score
- User Experience Suggestions
- Layout Improvement Tips

### 💡 Smart Suggestions

- SEO Recommendations
- Performance Improvements
- UI / UX Enhancements
- Portfolio Best Practices

## ⚙️ How It Works

1. Enter your portfolio website URL
2. Click "Audit Website"
3. View generated scores
4. Review recommendations
5. Improve your portfolio
